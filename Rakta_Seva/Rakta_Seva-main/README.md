# Rakta-Seva Connect

Rakta-Seva Connect is a "Life-Saving Bridge" Android application designed for voluntary blood donors and medical emergencies. It facilitates real-time availability of "Replacement Donors" at the taluka level, focusing on the "Golden Hour" of medical treatment.

## Features

- **Donor Registry**: Sign up with blood group and last donation date.
- **Emergency Request**: Dispatch blood requirements with urgency levels (Critical, High, Normal).
- **Proximity Alerts**: View nearby emergencies and get notified.
- **Admin Dashboard**: Monitor global requests and active donors.
- **Privacy First**: Phone numbers are only visible after a donation request is accepted.
- **Notification System**: Real-time updates for request acceptances and direct donor requests.

## Tech Stack

- **Language**: Kotlin
- **Architecture**: MVVM (Simulated with DataManager)
- **UI**: Material 3, ConstraintLayout, XML
- **Persistence**: SharedPreferences with GSON
- **Minimum SDK**: 24 (Android 7.0)
- **Target SDK**: 36

## Getting Started

### Prerequisites

- Android Studio (Ladybug or newer recommended)
- Android SDK 36
- Git installed on your system

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Rakta_Seva.git
   ```

2. **Open the project**:
   - Open Android Studio.
   - Click on `File` > `Open`.
   - Navigate to the folder where you cloned the project and click `OK`.

3. **Sync Project with Gradle**:
   - Android Studio will automatically prompt you to sync Gradle. If not, go to `File` > `Sync Project with Gradle Files`.

4. **Run the App**:
   - Connect an Android device or start an emulator.
   - Click the green `Run` button in Android Studio.
